# React Clone Twitch

## Description :
* Clone de certaines fonctionnalité de Twitch
* Searchbar pour rechercher un streamer
* Possibilité de regarder un streamer, de chatter via React Twitch Embed Video

## Technologies utilisées :
* React (Hooks)
* CSS 3
* API Twitch
